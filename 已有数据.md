# 已有数据总结

本项目目前包含以下四类主要数据：中老年疾病数据、养老机构数据、医保药品目录数据以及医疗保险数据。

## 1. 中老年疾病数据
- **文件路径**: `DataCleaned/中老年疾病目录/diseases.json`
- **数据格式**: JSON (对象列表)
- **数据内容**: 包含常见中老年疾病的详细信息。
- **字段说明**:
  - `id`: 疾病ID (如 "1")
  - `icd_code`: ICD编码 (如 "J15.0")
  - `name`: 疾病名称 (如 "肺炎杆菌肺炎")
  - `intro`: 疾病简介
  - `get_prob`: 患病概率 (如 "0.0002%")
  - `cure_dept`:就诊科室 (如 "内科呼吸内科")
  - `easy_get`: 易感人群 (如 "多见于老年人")
  - `get_way`: 传染性 (如 "有传染性")
  - `neopathy`: 并发症列表 (如 ["感染性休克", "脑疝"])
  - `drug`: 常用药物列表 (如 ["头孢氨苄胶囊", ...])
  - `cause`: 致病原因
  - `prevent`: 预防措施
  - `symptom`: 症状列表 (如 ["畏寒", "高热", ...])
  - `treat_detail`: 详细治疗方案（包括西医、中医等）
  - `nursing`: 护理建议

## 2. 养老机构数据
- **文件路径**:
  - `DataCleaned/养老机构数据/nursing_homes.csv` (CSV格式)
  - `DataCleaned/养老机构数据/nursing_homes.json` (JSON格式)
- **数据格式**: CSV / JSON (对象列表)
- **数据内容**: 北京地区的养老机构信息。
- **字段说明**:
  - `城市`: 所在城市 (如 "北京")
  - `名称`: 机构名称 (如 "北京朝阳区福祐养老院")
  - `性质`: 运营性质 (如 "民营", "公建民营")
  - `床位`: 床位数 (如 "300张")
  - `价格(元/月)`: 月费用 (如 "4500")
  - `特色服务`: 提供的特色服务 (如 "医养结合,邻近医院...")
  - `地址`: 详细地址

## 3. 医保药品目录数据
- **文件路径**: `DataCleaned/医保药品目录/medicine.json`
- **数据格式**: JSON (层级结构)
- **数据内容**: 包含西药、中成药等医保药品分类及具体药品信息。
- **结构说明**:
  - 顶层按药品大类划分（如 "西药部分"）。
  - `categories`: 药品的层级分类结构（包含代码、名称、层级等）。
  - `medicines`: 具体药品列表。
- **药品字段说明** (在 `medicines` 列表中):
  - `id`: 药品ID (如 "西药部分_4")
  - `name`: 药品名称 (如 "复方硼砂")
  - `sheet`: 所属大类 (如 "西药部分")
  - `reimbursement_category`: 医保报销类别 (如 "甲类", "乙类")
  - `category_code`: 大类代码 (如 "XA")
  - `subcategory_code`: 子类代码 (如 "XA01")
  - `subcategory_name`: 子类名称 (如 "口腔科制剂")
  - `dosage`: 剂型 (如 "外用液体剂")

## 4. 医疗保险数据
- **文件路径**: `DataCleaned/医疗保险数据/insurance_info.json`
- **数据格式**: JSON (对象列表)
- **数据内容**: 各类商业医疗保险产品信息。
- **字段说明**:
  - `产品名称`: 保险产品名称 (如 "蓝医保·长期医疗险")
  - `险种分类`: 保险类型 (如 "医疗险", "护理险")
  - `承保公司`: 保险公司 (如 "中国太平洋保险")
  - `承保年龄`: 适用年龄范围 (如 "出生满30天-70周岁")
  - `保障期限`: 保险期限 (如 "1年", "终身")
  - `产品描述`: 产品特点及保障范围简述
  - `价格`: 参考价格 (如 "18元/月起")
